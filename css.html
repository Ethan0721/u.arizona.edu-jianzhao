CSS-Multiple Classes for Single Element 05/19/2017

The above button used two classes on one single button element, what I am thinking this is useful is, for example, if I have ten buttons, they all have red background color, but 3 has blue text, 5 has yellow text, and 2 has orange text. This way is easier to implement.

<style> .class1{ background-color:red; } .class2{ color:yellow; } </style> <button class="class1 class2">button</button>
